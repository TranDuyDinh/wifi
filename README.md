# Wifi Scanning

Tool to scan Wi-Fi password by manually entering the password into the network.
> [!CAUTION]
> For educational purposes only, it cann't be used in practice.

## Features

- **Scan Wi-Fi Networks**: Discover all available Wi-Fi SSIDs in your area
- **Security Information**: Get authentication (ex. WPA2-Personal) and encryption (ex. CCMP) details for specific networks
- **Scan Wi-Fi password**: Using brute-force to scan Wi-Fi password

## Environment

- Python 3.x
- Windows operating system

## Usage

```bash
python ./main.py
```
